A simple website showing API usage for Code The Dream Pre-work Assignment
## How to Use

Visit [bumbl1na.github.io](https://bumbl1na.github.io/)

Or run locally:

1. Clone or download the repository
2. Open `index.html` in your browser


## Features

### Search for Dog Breeds

- Enter a breed name in the search bar  
  Examples: `labrador`, `poodle`, `husky`, `golden retriever`
- Click **Search** or press **Enter**
- View breed information and photos

### Random Dog Images

- Click **Get Random Dog**
- View 6 random dog photos in a responsive grid
- Click **Get More Dogs** to load more images

### Navigation for all pages

Use the navigation buttons to:
- Return to the home page
- Search for another breed
- View more random dogs
