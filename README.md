# ctd-dog-breed-api
A simple website showing API usage for Code The Dream Pre-work Assignment
HOW TO USE: 
1. Go to https://bumbl1na.github.io/ skip to step 2 or follow steps a-d below
  a. Clone the repository or save to your system 
      b.  **Open `index.html`** in your web browser
      c. - Double-click the `index.html` file, OR
      d. - Right-click and select "Open with" → Choose your browser

2. That's it! The website is now ready to use! 🎉
HOW TO USE THE SITE: 
Searching for a Dog Breed
 **Home Page** 
    **Type a breed name** in the search box  Examples: "labrador", "golden retriever", "poodle", "bulldog", "husky", "german shepherd"
    **Click the "Search" button** or press Enter on your keyboard
**View the results page** which shows:
   - Breed name at the top
   - All breed information in an easy-to-read list
   - 3 photos of the breed below the information
**Navigate back** by clicking "Back to Home" or "Get Random Dog →"

Getting Random Dog Images

**Home Page:**
   - Click the "Get Random Dog" button

   **View the results page**  
    click "Get Random Dog →"

 **View 6 adorable random dog photos** in a responsive grid

**Get More Dogs:**
   - Click "Get More Dogs" to load 6 new random photos

 **Navigate:**
   - "Back to Home" - Return to home page
   - "Search Breeds →" - Go search for a specific breed

Error Messages
If something goes wrong, you'll see a popup error message:

- **"Please enter a breed name!"** - You clicked search without typing anything
  - **Fix:** Type a breed name in the search box

- **"No breed found matching [name]"** - The breed name wasn't found
  - **Fix:** Try searching for a different breed. Common breeds: "labrador", "golden", "poodle", "bulldog", "husky"

- **"Error loading random dogs. Please try again."** - API connection issue
  - **Fix:** Check your internet connection and click "Try Again"
